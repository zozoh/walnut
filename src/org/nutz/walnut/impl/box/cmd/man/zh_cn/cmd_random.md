# 命令简介 

    `random` 命令输出一个随机数

# 用法

    random [-n] [-s 4] [11,23] 
    
    [11,23] 表示生成 11至23内的任意整数（包含）
    -n   表示是否在结尾输出空行
    -s   为随机数数量。默认为 4，这个模式下，每个随机数都是 0-9
    
    如果即不是 -s 模式，也没有声明整数范围，那么会生成一个 0.0 - 1.0 的随机浮点数
    
# 示例

    # 输出六个随机数
    demo@~$ random 6
    432134
    
    # 输出 18 至 100 内的随机整数
    demo@~$ random 18,100
    29
    
    # 输出 0.0 至 1.0 内的随机浮点数
    demo@~$ random
    0.341
