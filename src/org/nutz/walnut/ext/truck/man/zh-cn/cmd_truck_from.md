过滤器简介
======= 

`@from` 设置转换源
 

用法
=======

```bash
@from 
  [~/path/to]        # 【必】目录路径，必须为一个目录
  [-index dao(abc)]  # 指明索引映射方式，默认采用目录上的原始设定
  [-bm lbm(Tmp)]     # 指明桶管理器，默认采用目录上的原始设定
  [-match {...}]     # 指明一个过滤条件
  [-sort {ct:-1}]    # 指明一个排序方式，以便固定顺序分页
  [-limit 1000]      # 指明最大数据量（默认1000）
  [-skip 0]          # 指明跳过多少条数据开始
```


示例
=======

```bash
truck @from ~/account @to -index dao(abc)
```
