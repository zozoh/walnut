# 命令简介 

`truck` 命令用来转换数据。它可以将 `WnObj` 在已经定义的映射中转换

-------------------------------------------------------------
# 用法


```bash
truck 
  [-quiet]                       # 静默输出
  [-json]                        # 将结果输出为 JSON
  [-cqn]                         # 非静默输出的 JSON 格式
  [-ajax]                        # 非静默输出时，要 ajax 包裹
  [-t K1,K2...]                  # 表格输出
  [-bish]                        # 表格输出时的格式化
  [-out @{xx}]                   # 模板输出（默认的输出方式 @{id}）
  [[@filter filter-args...]...]  # 过滤器
```

它支持的过滤器有：

```bash
@from       # 设置转换源（桶或者索引）
@to         # 设置转换目标（桶或者索引）
@buf        # 设置缓冲区的大小
```
