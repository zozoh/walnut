# 命令简介 

`ti i18n_compare` 比较不同语言版本的 i18n，看看有什么疏漏

-------------------------------------------------------------
# 用法
 
```bash
ti i18n_compare
  [/path/to]        # i18n 文件所在的目录
```

-------------------------------------------------------------
# 示例

```bash
# 比较某目录下的多语言版本差异
demo:~$ ti i18n_compare i18n
```
