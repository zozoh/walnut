命令简介
======= 

`iotpower` 便携式物联网电路测量器的相关命令

用法
=======

```
iotpower activate
```

示例
=======

```bash
echo '{random:0x20210629,uuid:"0072002B0947303032333230",devid16:"205305D929663613121233F820210609"}' | iotpower activate
```
