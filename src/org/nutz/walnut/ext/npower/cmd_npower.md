命令简介
======= 

`npower` 用于控制智能插座
    

用法
=======

```
npower add          # 批量添加设备
npower list         # 查询设备,可分页
npower stat         # 查看设备状态
npower shadow       # 获取或修改设备影子
npower on           # 继电器设置为开,并设置为远程操作模式
npower off          # 继电器设置为关,并设置为远程操作模式
npower cron         # 继电器设置为关,并设置为定时器模式
```
