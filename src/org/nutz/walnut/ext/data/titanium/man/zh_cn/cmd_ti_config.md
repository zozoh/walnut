# 命令简介 

`ti config` 用来获取某个会话自定义的 Ti.Config 配置信息。
默认它会读取 `~/.ti/config.json` 文件，如果不存在则返回 `{}`

-------------------------------------------------------------
# 用法
 
```bash
ti config 
  [-name config.json] # [选] 配置文件名 
  [-cqn]              # [选]JSON 输出的格式化   
```

-------------------------------------------------------------
# 示例

```bash
# 根据对象得到可以创建的子对象类型列表
demo:~$ ti config
```
