# 过滤器简介

`@var` 为上下文设置变量

# 用法

```bash
wf @var
  [name]             # 指明变量名称，如果不指定名称
                     # 则会将目标对象所有的键作为变量名批量加入
                     # 如果加载目标是一个集合，
                     # 则会认为是 {list:[], count:8} 形式的对象
  [-f /path/to]      # 加载对象内容为变量
  [-o /path/to]      # 加载对象元数据为变量
```

# 示例

```bash
$demo> wf @var 
```
