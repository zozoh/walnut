# 过滤器简介

`@export` 将文件解压缩到指定目标

# 用法

```bash
unzipx abc.zip @export 
        [/path/to]   # 要输出的目标目录
        [-metas ]    # 输出目标的补充元数据集合，键为实体路径
                     # 元数据里的 nm 可以指定新名称
                     # 内容如果是 '{}' 则为元数据
                     # 否则被认为是路径
                     # 如果没值，则尝试从管线读取
        [-flat]      # 开启这个选项，则压缩包的实体直接输出到目标目录
                     # 否则如果相对路径可能会创建出子目录。
        [-buf 8192]  # 写入文件时的缓冲，默认8K
        [-quiet]     # 静默输出，否则会输出生成的对象 json 列表
        [-cqn]       # 非景默输出时,的json 格式化方式
        [-v]         # 显示显示导出的细节

```

# 示例

```bash
# 列出压缩包中的所有实体
unzipx abc.zip @export ~/tmp/ 
```
