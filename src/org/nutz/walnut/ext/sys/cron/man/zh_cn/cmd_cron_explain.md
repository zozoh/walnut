# 命令简介 

`cron explain` 解释一组`cron`表达式的执行结果

# 用法

```bash
cron explain
  [WnCron...]     # 【必】指定多个定期任务表达式
  [-today now]    # 指定一个日期，默认为 now 表示当天
  [-lang zh-ch]   # 多国语言，默认为当前系统语言 zh-cn
  [-slot 24]      # 时间槽的数量，默认为24表示24小时，
                  # 如果精度到分钟，应该是 1440
```
