# 命令简介 

`cron preview` 预览指定定期任务的计划表

**本命令如果是非`root`组管理员执行，则只能预览自己的任务**

# 用法

```bash
task preview
  [ID...]          # 指定一个或者多个任务的 ID
  [-u xxx]         # 仅列出某个用户的任务
  [-skip 0]        # 跳过多少记录
  [-limit 0]       # 最多列出多少记录
  [-today now]     # 指定预览日期，默认是当天
  [-empty]         # 空时间槽也要显示
  [-slot 24]       # 时间槽的数量，默认为24表示24小时，
                   # 如果精度到分钟，应该是 1440
```
