# 命令简介 

`dsync @tree` 构建或加载对象的索引树

# 用法

```bash
dsync @tree
  [{SHA1}]       # 【选】指定包的指纹，根据配置信息以及指纹能确定唯一的包名 
                 # 如果没有指定，则会操作HEAD索引树，否则会从缓存目录里加载
                 # 对应的索引树
  [-f -force]    # 仅仅是对HEAD操作的时候才有效。表示强制要重新建立索引树。
                 # 否则会自动比对索引树文件与目标目录的同步时间，如果不相同
                 # 会重新建立索引树
  [-I -items]    # 确保为所有的索引项目都加载了 WnObj 数据
  [-H -head]     # 表示仅仅是加载现有的HEAD索引树
```
