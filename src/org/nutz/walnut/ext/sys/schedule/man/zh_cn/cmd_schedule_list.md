# 命令简介 

`schedule list` 列出所有的分钟计划任务

**本命令如果是非`root`组管理员执行，则仅会查看自己任务**

# 用法

```bash
schedule list
  [2021-09-11] # 指定一个日期，默认是当天
               # 同时支持相对时间，譬如
               #  -1d 表示昨天
               #  1d  表示明天
  [[0,1440)]   # 时间槽的区间，默认为 [0,1440) 表示全天
  [-u xxx]     # 仅列出某个用户的任务
  [-t ...]     # 按表格显示的任务字段，否则就是按JSON
  [-cqn]       # JSON 格式化参数
```
