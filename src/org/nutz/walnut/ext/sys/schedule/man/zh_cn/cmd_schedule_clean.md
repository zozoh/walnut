# 命令简介 

`schedule clean` 主动清除过期的分钟计划

**!!!本命令需要`root`组管理员权限才能执行**

# 用法

```bash
schedule clean
  [2021-09-11]    # 指定一个日期，默认是当天
                  # 同时支持相对时间，譬如
                  #  today-1d 表示昨天
                  #  today+1d 表示明天
  [-u xxx]        # 仅针对某个用户的任务
  [-skip 0]       # 跳过多少记录
  [-limit 0]      # 最多列出多少记录
  [-t ...]        # 按表格显示的任务字段，否则就是按JSON
  [-cqn]          # JSON 格式化参数
```
