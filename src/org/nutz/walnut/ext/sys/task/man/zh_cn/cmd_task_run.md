# 命令简介 

`task run` 主动执行任务堆栈里的一个或者多个任务

**!!!本命令需要`root`组管理员权限才能执行**

# 用法

```bash
task run
  [ID]         # 指定一个任务的 ID
  [-u xxx]     # 指定任务的所属用户
  [-limit 1]   # 最多运行多少条命令，默认 1
  [-remain]    # 返回任务堆栈里剩余任务摘要
```

本命令执行结果，会返回当前任务堆栈里剩余的任务对象列表：

```json
{
  "count" : 1,
  "tasks" : [{
    "id"   : "56y1..a8e2",
    "user" : "zozoh",
  }]
}
```

这里需要指出的是，如果声明了`ID`则返回大多数时候是空的，因为指定的`ID`已经被弹出执行了。
如果声明了`-u`则只会返回所属用户其余的命令。
