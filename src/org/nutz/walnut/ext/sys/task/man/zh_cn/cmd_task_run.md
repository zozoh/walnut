# 命令简介 

`task run` 主动执行任务堆栈里的一个或者多个任务

**!!!本命令需要`root`组管理员权限才能执行**

# 用法

```bash
task run
  [ID]               # 指定一个任务的 ID
  [-u xxx]           # 指定任务的所属用户
  [-skip 0]          # 跳过多少记录
  [-limit 0]         # 最多处理多少记录，默认 0 表示不限制
  [-json]            # 指定按照 JSON 格式输入出内容
  [-cqn]             # JSON 格式化参数
```
