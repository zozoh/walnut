# 命令简介 

`datex @time` 修改上下文时间

# 用法

```bash
datex @time 
  [$time]     # 设置上下文的时间
              # 支持正负整数，表示一天的绝对毫秒数
              # 也可以是一个时间字符串，譬如 05:19 或者 08:12:56.666
  [-i $time]  # 偏移上下文时间，支持毫秒，或者时间字符串
```
# 示例

```bash
# 直接将时间设置为 00:00:00
datex @set 2023-09-21 @time 0

# 直接将时间设置为 00:10:00
datex @set 2023-09-21 @time 00:10:00

# 直接将时间设置为 23:59:59.999
datex @set 2023-09-21 @time -1

# 直接将时间设置为 23:59:59.000
datex @set 2023-09-21 @time -1000
```
