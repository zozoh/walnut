# 过滤器简介

`@find` 寻找到一组符合要求的节点，并记入上下文

# 用法

```bash
dom @find 
  [selector ...]   # 多个选择器，为或的关系
                   # 现在仅支持
                   #  - 标签名选择器
                   #  - 属性选择器
                   #  - 类选择器
                   #  - ^... 的为正则表达式形式的标签选择器
```

# 示例

```bash
dom @html a.html @find .my-div
```
