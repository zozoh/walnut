# 过滤器简介

`@tree` 将上下文对象归纳成一棵树

# 用法

```bash
o @tree
  [-depth 0]      # 将每个对象向下读取多少层，默认 0，表示不读取
  # 默认的，会自动归纳为一个公共的根节点
  # 但是通过这个参数，你可以指定一个公共根节点
  # 当然，这个节点必须是在对象树中的，否则无用
  [-top /path]
  # 强制设定文件系统的根节点为树的根节点
  [-root]
```

# 示例

```bash
# 将某个文件夹向下展开 2 层
$demo:> o path/to/dir @tree -depth 2

# 将一组给定的文件对象，归纳为一棵树
$demo:> o f1 f2 f3 @tree
```
