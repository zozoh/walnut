# 过滤器简介

`@load` 读取报文文本，并加载到上下文中，支持模板渲染

# 用法

```bash
edi @load 
  [Path]             # 【必】报文文件路径
  [-render {..}]     # 指明报文文件是模板，那么可以用这个
                     # 参数传递模板变量。
                     # 如果参数是  '{...}' 形式的文本则认为是JSON
                     # 否则认为是一个路径
                     # 如果没有值，则尝试从管道读取
  
```

# 示例

```bash
edi @load ~/demo.edi.txt -render ~/vars.json 
```
