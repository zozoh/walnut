# 过滤器简介

`@search` 从服务器收取邮件
 

# 用法

```bash
imap @search
  [0:{Flag},{Flag}]    # 标记，多个标记半角逗号分隔，大小写不敏感
                       # 开头的 0 表示未设置标记
                       # 如果是 1 表示收取所有加入本标记的邮件
                       # 默认未 0:SEEN 表示所有未读邮件
                       # 多个条件形成的是 OR 的关系
```

**邮件支持的标记**

Flag       | Description
-----------|---------------------
`ANSWERED` | 邮件是否已回复。
`DELETED`  | 邮件是否需要删除。
`DRAFT`    | 邮件是否为草稿。
`FLAGGED`  | 邮件是否为回收站中的邮件。
`RECENT`   | 邮件是否为新邮件。
`SEEN`     | 邮件是否已被阅读。
`USER`     | 是否支持用户自定义标记(应用程序只读)
