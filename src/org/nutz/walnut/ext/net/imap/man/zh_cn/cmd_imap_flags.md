# 过滤器简介

`@flags` 设置上下文中的邮件标记
 

# 用法

```bash
imap @flags
  [0:{Flag},{Flag}]    # 标记，多个标记半角逗号分隔，大小写不敏感
                       # 开头 0 表示将指定标记设置为 false
                       # 开头 1 表示将指定标记设置为 true
```

**邮件支持的标记**

Flag       | Description
-----------|---------------------
`ANSWERED` | 邮件是否已回复。
`DELETED`  | 邮件是否需要删除。
`DRAFT`    | 邮件是否为草稿。
`FLAGGED`  | 邮件是否为回收站中的邮件。
`RECENT`   | 邮件是否为新邮件。
`SEEN`     | 邮件是否已被阅读。
`USER`     | 是否支持用户自定义标记(应用程序只读)
