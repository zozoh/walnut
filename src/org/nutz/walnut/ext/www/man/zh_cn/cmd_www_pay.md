命令简介
======= 

`www pay` 根据订单创建支付单

用法
=======

```bash
www pay
    [ID]      # 【必】订单的 ID
    #-----------------------------------------------
    # 输出的JSON数据的配置信息
    [-cqn]    # JSON 输出的格式化方式
    [-ajax]   # 开启这个选项，则输出为 ajaxReturn 的包裹
```

示例
=======

```bash
# 支付一个订单
demo:> www pay 47ra..9q1a
```
