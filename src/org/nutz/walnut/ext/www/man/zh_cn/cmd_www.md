命令简介
======= 

`www` 用来处理用户域名映射和发布目录的相关操作
    

用法
=======

```
www query       # 【废弃】查询域名映射的信息列表
www add         # 【废弃】添加一个新的域名映射
www rm          # 【废弃】移除一个域名映射
www renew       # 【废弃】处理域名的续费成功的后续操作
www conf        # 【废弃】得到域名映射方面的配置文件信息(JSON)

www captcha      # 验证码
www checkme      # 获取当前会话信息
www login        # 登录/绑定手机/注册
www logout       # 注销会话 
www resetpasswd  # 重置密码
```
