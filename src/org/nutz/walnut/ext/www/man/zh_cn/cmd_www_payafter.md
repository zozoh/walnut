命令简介
======= 

`www payafter` 检查一个订单的支付状态，如果支付完成（成功或者失败）则自动同步订单状态

用法
=======

```bash
www payafter
    [OrderID]       # 【必】订单的 ID
    #-----------------------------------------------
    # 输出的JSON数据的配置信息
    [-cqn]    # JSON 输出的格式化方式
```

示例
=======

```bash
# 支付一个订单
demo:> www payafter 4r..wa 
```
