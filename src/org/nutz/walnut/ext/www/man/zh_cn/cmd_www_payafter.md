命令简介
======= 

`www payafter` 同步一个订单的支付状态，如果支付完成（成功或者失败）则自动同步订单状态

用法
=======

```bash
www payafter
    [OrderID]       # 【必】订单的 ID
    #-----------------------------------------------
    # 启用了这个参数后，如果订单支付完成，
    # 会对标准订单（A订单）调用  buy it 命令以便从购物车删除对应商品。
    # 如果没有指定名称，
    # 仅仅是 -basket参数，则相当于 -basket _buy
    # 否则，相当于指定购物车配置文件名(buy it -conf xxx) 
    [-basket _buy]
    #-----------------------------------------------
    # 输出的JSON数据的配置信息
    [-cqn]    # JSON 输出的格式化方式
```

示例
=======

```bash
# 支付一个订单
demo:> www payafter 4r..wa 
```
