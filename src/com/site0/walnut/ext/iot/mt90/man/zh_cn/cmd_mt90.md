# 命令简介 

    `mt90` 处理MT90追踪设备的相关逻辑

# 用法

```
mt90 parse              # 处理设备上传的定位信息
mt90 ply_update         # 使用设备定位信息更新选手数据
mt90 mock               # 模拟设备上传
mt90 map_match          # 赛事轨迹匹配及预测
mt90 toimage            # 把轨迹画成图片
mt90 eta                # 根据选手轨迹,预估到达时间,同时进行路线匹配
```
