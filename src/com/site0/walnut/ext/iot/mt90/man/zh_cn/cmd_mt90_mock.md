# 命令简介 

    `mt90 mock` 模拟设备上传

# 用法

```
mt90 mock 
		[id:xxxx] # 设备文件的id
		[imei]    # 设备的imei, 与设备文件id只能二选一,而且必须填一个
		[-y]      # 真正开始模拟, 否则仅输出模拟概要
		[-start XX] # 起始点的百分比, 默认是0,即赛事起点,范围是0-99
		[-end XX]   # 终点的百分比, 默认是1000,即赛事终点,范围是1-100
		[-t XX]        # 发送间隔,单位秒,默认为5秒一次
		[-c XX]        # 需要发送的轨迹点个数,通常根据start和end得出
		[-url xxx]     # 目标网址,默认是本机,慎用
```
