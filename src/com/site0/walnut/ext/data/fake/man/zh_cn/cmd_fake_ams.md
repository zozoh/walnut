# 过滤器简介

`@ams` 生成绝对毫秒数

# 用法

```bash
fake [N] @ams
  [2021-12-19]    # 开始日期时间，也可以是一个绝对毫秒数
                  # 默认为 today
  [5m]            # 一个毫秒数的表达式，标识一个随机区间
                  # 默认为 5m
  [-as AMS]       # 输出的类型
                  # - "AMS" 表示输出绝对毫秒数【默认】
                  # - "yyyy-MM-dd"  其他的表示日期格式化
```

# 示例

```bash
# 输出连续三个绝对毫秒数，并按照指定日期时间格式输出
demo> fake 3 @ams today 5m -as "yyyy-MM-dd'T'HH:mm:ss.SSS"
2021-05-16T00:01:00.729
2021-05-16T00:06:50.763
2021-05-16T00:11:40.054
```
