# 过滤器简介

`@load` 读取模板内容


# 用法

```bash
tmpl @load  
  [-f /path/to]      # 内容来自文件
  ["xxx${x}xxx"]     # 模板内容
  # 指定模板符号形式，是一个数组
  # 0) '$$' 逃逸字符
  # 1) '${' 开始字符
  # 2) '}'  结束字符
  # 表示遇到文本 ${xxx} 则取 xxx 作为一个模板动态符号
  # $${xxx} 可以逃逸，相当于文本 "${xxx}"
  # 这里开始字符有个约定,最后一个字符 '{' 将会压栈，必须有一个
  # 结束字符 '}' 才会导致退栈。
  # 也就是说动态符号 `${{x:100}}` 解析后，会得到 `{x:100}`
  # 即，你可以在动态符号内嵌套多层括号
  [-by '["$$","${","}"]']
```

# 示例

```bash
# 从文件里获得
tmpl @load -f ~/mytmpl.txt
# 从参数里获得
tmpl @load 'a${x}b'
# 从管道获得
cat ~/mytmpl.txt | tmpl @load
```
