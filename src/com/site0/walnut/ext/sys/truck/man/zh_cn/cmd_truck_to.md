过滤器简介
======= 

`@to` 设置转换目标，同时确定转换模式

- `INDEXER` 仅转换索引
- `BM` 仅转换桶
- `BOTH` 转换索引和桶
 

用法
=======

```bash
@to 
  [~/path/to]        # 【选】目录路径，默认采用 @from 的路径
                     # 同时也会确定这个目录默认管理器就是目标管理器
  [-mode INDEXER]    # 转换方式，默认为 INDEXER
  [-noexists]        # 打开这个开关，只有不存在的，才插入（会预先检查，所以慢一些）
  [-genid]           # 转换的数据，需要新生成 ID
```
