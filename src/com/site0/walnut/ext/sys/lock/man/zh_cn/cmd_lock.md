# 命令简介

`lock` 用来管理全局的锁

用户可以用它来查看系统锁服务的情况，脚本可以用它来申请释放锁


# 用法

```
lock [-cqn -G] [@try|@get|@free|@list]
[-cqn]    # 输出信息 Json 格式化方式
[-G]      # 全局模式，默认的是「域模式」
          # 即，你操作的所有锁名，都会默认加上域名前缀
          # 譬如锁名是 `abc`，相当于你写了 `myDomain:abc`
```

# 过滤器列表

```bash
@try       # 尝试申请一个锁
@get       # 获取一个锁
@free      # 释放一个锁
@list      # 列出符合条件的锁信息
```
