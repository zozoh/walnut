# 采用git的文件观念，还是Linux的文件观念？

 - 采用 Linux 的文件观念。版本控制不是必须的。
 - 文件索引存放在MongoDB里。也包括所有的元数据和标签
 - 文件的索引根据 SHA1 可以找到文件的裸内容（和git一样）
 - 文件的权限和钩子触发由 API 层处理，核心层不管
