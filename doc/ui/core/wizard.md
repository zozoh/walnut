# 控件概述

向导控件将聚合一组 ZUI 完成对一个 obj 的各个字段填充的过程

# 如何创建实例

```
new WizardUI({
    title : "i18n:xxx"   // 整个向导的标题
    
    // 向导标题类型
    //  - single : 只显示当前步骤标题
    //  - all    : 显示所有步骤，并高亮当前步骤
    // 默认为 single
    headMode : "single|all"
    //..........................................
    // 「选」向导初始的数据
    // 也可以在 render 的会调里动态设置 
    data : {..}
    //..........................................
    // 「选」初始步骤
    //  - "step1"          # 字符串表示确定一个步骤
    //  - {c}F(data):step  # 函数可以动态的返回
    // 也可以在 render 的会调里动态设置 
    startPoint : "step1"
    //..........................................
    // 定义了各个步骤的逻辑和显示方式
    //  - 键为步骤名
    //  - 值为步骤的具体设置
    steps : {
        "step1" : {
            icon : '<..>',      // 「选」图标
            text : 'i18n:xxx',  // 「选」文字描述
            
            // 界面控件(必须支持getData/setData)
            uiType : "name/to/ui",  
            uiConf : {..}
            
            // 当前步骤支持的操作，如果为空，则不显示
            actions : [{
                icon : '<..>'      // 按钮图标
                text : 'i18n:xxx'  // 按钮文字
                
                // 执行后进入哪一步，可以是字符串或者函数
                jumpTo : 'step2' | {c}F(data):step
            }]
        },
        "step2" : {
            // 另一个步骤的配置
        }
    },
    //..........................................
    // 各个回调函数的上下文，默认为 WizardUI 实例自身
    context : UI
    //..........................................
}).render(function(){
    // 这里你可以更灵活的设置初始数据
    this.setData({..});
    
    // 也可以更灵活的设置初始步骤
    this.setStartPoint("step1");
});
```
