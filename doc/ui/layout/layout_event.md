----------------------------------
# 事件·概述

```bash
area:show      # 显示某个 area (包括box/tab) 完毕（如果有动画，也在动画加载完毕后）
area:ready     # 某个area内部全部控件加载完毕
```

如果一个 `box/tab` 显示了，其事件调用顺序应该是

```bash
area:show
# 如果有的话，就异步加载所有的控件
# 所有加载完毕以后会调用
    |-- area:ready
```

----------------------------------
# 事件·对象

任何一个布局事件，其对象如下:

```js
{
    area : jQuery,      // 表示一个事件所在的区域
    key  : "chute",     // 事件所在区域的 key
    type : "box:show",  // 事件的类型
    // 如果是 ready 事件，那么会有下面的段表示本区域下所有的
    // UI 加载实例映射表
    uis  : {
        "gasA" : UI,
        "gasB" : UI,
    }
}
```
